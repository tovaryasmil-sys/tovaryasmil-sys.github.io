Cómo poner esta página online gratis con GitHub Pages

1. Crear el repositorio en GitHub

Crea un repositorio llamado tovaryasmil-sys.github.io para que la URL sea https://tovaryasmil-sys.github.io. Sube este archivo como index.html.

2. Activar GitHub Pages (hosting gratis)

En el repositorio, ve a Settings → Pages, elige la rama main y la carpeta raíz /. Guarda y espera 1–3 minutos hasta que la URL quede activa con HTTPS.

3. Actualizar la web desde Replit (opcional)

Si generas el código en Replit, conecta el proyecto con tu repo de GitHub y haz commit + push cada vez que quieras publicar cambios. GitHub Pages se actualiza solo.

4. Añadir tu agente de IA (chatbot)

Regístrate en un proveedor de chatbot y copia el script de incrustación que te den. Pégalo justo antes de la etiqueta de cierre </body> en este archivo y súbelo a GitHub para tener tu asistente activo 24/7.

5. Personalizar y escalar

Cambia textos, colores y secciones editando el HTML/CSS. Cuando estés listo, compra un dominio propio e intégralo con GitHub Pages sin cambiar de hosting.
